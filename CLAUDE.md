# V-Genesis - 動画創成システム

## プロジェクト概要
AIエージェントを活用した汎用動画制作自動化パイプライン。
YouTube解説動画・ショート動画・プロモーション動画に対応。

## 言語設定
- 主言語: 日本語
- 副言語: 英語
- タイムゾーン: JST

## パイプライン（7フェーズ）
1. **Research** → analyst, explore, ux-researcher, document-specialist, scientist
2. **Script** → writer, prompter, localization, critic, code-simplifier
3. **Visual** → designer, accessibility
4. **Production** → executor, debugger, tracer, optimizer, data-pipeline
5. **QA** → qa-tester, code-reviewer, verifier, security-reviewer, compliance, risk-assessor
6. **Publish** → api-designer, devops, git-master, database
7. **Post-Publish** → monitor, mobile, migrator, refactorer

## エージェントルーティング
- トピックリサーチ → `analyst`
- 脚本作成 → `writer`
- ビジュアル設計 → `designer`
- 品質検証 → `verifier`
- SEO最適化 → `prompter`
- パイプライン設計 → `architect`
- バグ調査 → `debugger`
- パフォーマンス → `optimizer`

## ディレクトリ規約
- `projects/{project-id}/` - 各動画プロジェクトのワークスペース
- `templates/` - 再利用可能なテンプレート
- `config/` - パイプライン設定
- `assets/` - 共有アセット

## 品質ゲート
- スクリプト品質スコア: 8/10以上
- アクセシビリティ: WCAG 2.1 AA準拠
- SEOスコア: 75以上
- サムネイルCTR予測: 5%以上

## 禁止事項
- 著作権素材の無断使用
- 誤情報・虚偽コンテンツ
- プラットフォームガイドライン違反
