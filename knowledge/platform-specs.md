# プラットフォーム仕様ガイド

## YouTube

| 項目 | 仕様 |
|---|---|
| 推奨解像度 | 1920x1080 (FHD) / 3840x2160 (4K) |
| アスペクト比 | 16:9 |
| 最大ファイルサイズ | 256GB |
| 対応形式 | MP4, MOV, AVI, MKV, etc. |
| 最大動画長 | 12時間（認証済みアカウント） |
| タイトル上限 | 100文字 |
| 説明文上限 | 5000文字 |
| タグ上限 | 合計500文字 |
| サムネイル | 1280x720 px, JPG/PNG, 2MB以下 |

## YouTube Shorts

| 項目 | 仕様 |
|---|---|
| 推奨解像度 | 1080x1920 |
| アスペクト比 | 9:16 |
| 最大長さ | 60秒 |
| タイトル上限 | 100文字 |

## TikTok

| 項目 | 仕様 |
|---|---|
| 推奨解像度 | 1080x1920 |
| アスペクト比 | 9:16 |
| 最大長さ | 180秒（アカウントによる） |
| キャプション上限 | 2200文字 |
| ハッシュタグ推奨数 | 3-5個 |

## Instagram Reels

| 項目 | 仕様 |
|---|---|
| 推奨解像度 | 1080x1920 |
| アスペクト比 | 9:16 |
| 最大長さ | 90秒 |
| キャプション上限 | 2200文字 |
| ハッシュタグ上限 | 30個 |

## 字幕・アクセシビリティ
- SRT形式を標準採用
- 日本語・英語の2言語対応
- 1行あたり最大42文字（日本語）/ 84文字（英語）
- 表示時間: 最低1秒、最大7秒/行
